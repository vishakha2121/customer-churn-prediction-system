## 🚀 Overview

An end-to-end **AI-powered Customer Churn Prediction System** that helps businesses identify customers at risk of churning and provides personalized retention strategies. The system achieves **85% accuracy** using Random Forest algorithm and segments customers using **K-Means clustering**.

### 🎯 Key Features

- 🔮 **Churn Prediction** - Predict customer churn probability with 85% accuracy
- 📊 **Customer Segmentation** - K-Means clustering for targeted marketing
- 💎 **Retention Strategies** - AI-powered personalized retention offers
- 📈 **ROI Simulation** - What-if analysis for business decisions
- 🎨 **Interactive Dashboard** - Real-time analytics with React
- 📱 **Responsive Design** - Works on desktop and mobile devices

### 📊 Model Performance

| Metric | Score |
|--------|-------|
| Accuracy | 85.7% |
| Precision | 82.3% |
| Recall | 79.8% |
| F1 Score | 81.0% |
| ROC-AUC | 0.88 |
